Configure the prices for each resource type. Users will pay these prices when purchasing resources.

Number of coins
Amount of RAM in GB

CPU Pricing

Number of coins
Percentage of CPU (can exceed 100%)
Example: 1 coin = 1% CPU, or 2 coins = 5% CPU

Storage Pricing

Number of coins
